Docker问题(一)：bash: vi: command not found

在使用docker时，有时候我们需要编辑配置文件，需要使用vim或者vi命令,但是会发现：

bash: vi: command not found

因为vim没有安装，这个时候需要我们使用apt-get install vim命令安装vim

会出现如下问题：

此时执行apt-get update，然后再次执行apt-get install vim即可成功安装vim。
